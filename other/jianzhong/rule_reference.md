# 简中规则定义

## 牌张与基本流程

- 四人麻将，使用万、筒、索和七种字牌，共 136 张。
- 无花牌、无王牌区、无定缺、无查叫、无错和流程。
- 合法牌型包括标准四面子一雀头、七对和十三幺。
- 合法牌型即使没有任何计分番种，也允许以 0 分和牌。
- 吃只允许点炮者的固定下家；碰和杠按通常优先级处理。
- 吃、碰后必须出牌；明杠、暗杠、加杠后补牌并重新检查手牌动作。
- 最后一张弃牌只允许和牌响应，不允许吃、碰、杠。

## 终局流程

建房可选择首家、第二家或第三家和牌后结束。第二/第三家模式中，已和玩家退出本局。一次弃牌或抢杠可能产生多家独立确认的和牌；点击顺序不影响结算顺序。

多响按点炮者之后的固定座次排序。终局阈值只在本次所有确认赢家完成结算后检查。

## 过和限制

- 放弃某张弃牌的和牌机会后，在自己下次出牌前不能再点和同一牌。
- 玩家出牌时，原限制集合被替换为自己刚打出的牌。
- 自摸不受此限制；抢加杠和按点和处理。

## 庄家与局数

每局固定轮庄，不因和牌者、点炮者或流局改变。达到配置局数后结束整场。

## 计分支付

设和牌番值为 X：

- 点和或抢杠和：责任玩家支付 6X。
- 自摸且三名未和玩家仍在：每人支付 2X。
- 自摸且两名未和玩家仍在：每人支付 3X。
- 自摸且一名未和玩家仍在：该玩家支付 6X。
- 0 分合法和牌不产生分数转移。

局中续打时，真实番种、手牌、和牌张与分数变化延迟到本局终局展示。

## 信息隐藏

- 暗杠者看到真实牌，其他玩家只收到隐藏后的暗杠数据。
- 局中和牌不向其他玩家公开完整手牌、番种、分数和自摸牌。
- 重连按相同可见性规则重建状态。
- 终局牌谱保存真实数据，用于结算与回放。
