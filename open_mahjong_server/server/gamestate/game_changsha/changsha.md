# 长沙麻将经典双鸟规则

- 使用万、筒、条 1-9，共 108 张牌。
- 允许吃上家牌，允许碰、杠和抢杠胡。
- 小胡要求 4 组面子和 258 将；经典基础分保持小胡 1/2、大胡 6/7。
- 大胡支持碰碰胡、将将胡、清一色、全求人、七小对、豪华七小对，以及天胡、地胡、海底、杠上开花、杠上炮、抢杠胡等情境牌型。
- 扎鸟张数可选 0、1、2、4，可选择从本局开始庄座位 0 或胡牌玩家起算。
- 海底牌先盖放在桌面中央。玩家选择后翻开并进入该玩家牌河；该玩家能胡时按海底自摸结算，否则视为打出海底牌，只允许其他玩家荣和或跳过。
- 开杠补出的每张牌独立判断和牌。多张补牌可胡时，手牌牌型只计算一次，并按可胡张数累计杠上开花。
- 开杠补牌阶段禁止改动原手牌；不能和牌时只能打出服务端指定的补牌。
- 吃、碰、明杠统一使用 `cut_from_player` 和 `cut_tile` 标识被认领的弃牌来源和牌值。
