# 长沙麻将经典双鸟规则

- 使用万、筒、条 1-9，共 108 张牌。
- 允许吃上家牌，允许碰、杠和抢杠胡。
- 小胡要求 4 组面子和 258 将。
- 大胡支持碰碰胡、将将胡、清一色、全求人、七小对、豪华七小对、双豪华七小对、三豪华七小对，以及天胡、地胡、海底、杠上开花、杠上炮、抢杠胡等情境牌型。
- 全求人按四组副露和单钓成立，不要求 258，自摸和点炮都可以成立。
- 默认使用区分庄闲的经典基础分：闲家小胡 1 分、庄闲相关小胡 2 分，闲家大胡 6 分、庄闲相关大胡 7 分。
- 可选择“不区分庄闲”计分，默认小胡 2 分、大胡 8 分，并可在创建房间时自定义。
- 大胡按牌型权重累加。普通大胡权重为 1，豪华、双豪华、三豪华七小对权重分别为 2、3、4。
- 扎鸟张数可选 0、1、2、4。定庄扎鸟从本局开始庄座位 0 起算，赢家扎鸟从胡牌玩家起算；庄只影响鸟的归属和翻倍，不改变“不区分庄闲”模式的基础分。
